# Flask기반의 웹사이트
  이 웹사이트는 자취방을 구하러 다닐 때, 편하게 구할 수 있도록 하기 위해 제작된 웹사이트입니다.
  100% 파이썬과 플레스크로 제작하였습니다.
  
  CSS를 사용하지 않아서 굉장히 투박하지만, 플레스크의 기초적인 원리를 실습학 위해 제작하였습니다.


# Youtube 강의영상 참조

  이 영상을 통해 파이썬과 기초적이 플레스크에 대해서 익히실 수 있습니다.
